NGO Better Together

NGO Better Together is a web application built with React, Node.js, and PostgreSQL to streamline NGO and beneficiary management. It allows importing and generating Excel and PDF reports, provides full CRUD operations for beneficiaries, and includes automated email notifications using Nodemailer. With JWT authentication ensuring secure access, the app balances functionality and security.

I collaborated with clients to tailor features to their needs, enhancing my communication and problem-solving skills. Using GitHub for version control, I gained valuable experience in full-stack development, project management, and team collaboration.
